calculates the noise floor and the envelope values from the energies, depending on framing and stereo mode

FDKsbrEnc_extractSbrEnvelope is the main function for encoding and writing the envelope and the noise floor. The function includes the following processes:

-Determine time/frequency division of current granule. -Sending transient info to bitstream. -Set amp_res to 1.5 dB if the current frame contains only one envelope. -Lock dynamic bandwidth frequency change if the next envelope not starts on a frame boundary. -MDCT transposer (needed to detect where harmonics will be missing). -Spectrum Estimation (used for pulse train and missing harmonics detection). -Pulse train detection. -Inverse Filtering detection. -Waveform Coding. -Missing Harmonics detection. -Extract envelope of current frame. -Noise floor estimation. -Noise floor quantisation and coding. -Encode envelope of current frame. -Send the encoded data to the bitstream. -Write to bitstream.

Parameters
sbrHeaderDatahandle to config data
